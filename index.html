Glossário de html e css

Bem vindo, selecione abaixo qual deseja ver

HTML

O HTML é responsável por estruturar o conteúdo de uma página. Ele define os elementos que aparecem no site, como textos, títulos, imagens, links, listas, formulários, tabelas, entre outros. Podemos pensar no HTML como o esqueleto da página, onde cada parte tem uma função específica. Por exemplo, uma tag < p > representa um título principal, enquanto a tag < h1 > representa um parágrafo de texto.


CSS

O CSS é usado para estilizar esse conteúdo, ele cuida da aparência visual da página. Define quais elementos estarão na página (como textos, imagens e botões), o CSS define como esses elementos vão parecer para o usuário. É possível escolher as cores do fundo e dos textos, o tamanho das fontes, o espaçamento entre os elementos, o posicionamento de cada parte da página, e aplicar efeitos visuais, como sombras, transições e animações.
