O Display é um comando dos mais básicos, mas muito preciso, digo de precisãoç. Ele tem várias alternativas, a mais comum que falaremos aqui é flex

O flex faz com que um, ou mais elementos de uma conteiner, div, main... (seja oque for do tipo), fique alinhdos

quando não colocamos o display flex, os elementos ficam empilhados da maneira que foram escritos

Com o flex, os elementos se alinham em uma linha

O justify-content é o código que faz com que o(s) elemnto(s) de uma div após estarem num formanto de linha [ver códido display flex], fiquem em diferentes posições

    O justify-content contém vários subs do comando para cada ocasião que precisar Ele pode conter:
  • flex-start (forma padrão)
  • center
  • flex-end
  • space-around
  • space-evenly
  • space-between

flex-start → Alinha todos os itens no início do contêiner (lado esquerdo em layouts horizontais).

center → Centraliza todos os itens no meio do contêiner.

flex-end → Alinha todos os itens no final do contêiner (lado direito em layouts horizontais).

space-around → Distribui os itens com espaços iguais ao redor de cada um, ou seja, metade do espaço nas bordas externas e espaço inteiro entre itens.

space-evenly → Distribui os itens com espaços iguais tanto nas bordas externas quanto entre cada item.

space-between → Coloca o primeiro item colado no início e o último colado no final, distribuindo o espaço restante igualmente entre os itens do meio.

A propriedade box-shadow no CSS permite criar sombras ao redor de elementos, dando profundidade e destaque no layout. É muito usada para criar efeitos visuais que simulam iluminação, flutuação ou relevo.

box-shadow: deslocamentoX deslocamentoY raio-desfoque cor;

deslocamentoX → posição da sombra no eixo horizontal (valores positivos empurram para a direita, negativos para a esquerda).

deslocamentoY → posição no eixo vertical (positivos para baixo, negativos para cima).

raio-desfoque → define o quanto a sombra será borrada (quanto maior, mais suave).

cor → define a cor e a opacidade da sombra (normalmente se usa rgba() para transparência).

Opcionalmente, você pode adicionar:
spread (antes da cor) → aumenta ou diminui o tamanho da sombra.

inset → transforma a sombra em interna, dentro do elemento.

Um gradiente linear é uma transição suave de cores em linha reta, que pode ser horizontal, vertical, diagonal ou em qualquer ângulo que você escolher.

Parametros

direção → pode ser:
to right (esquerda → direita)
to bottom (cima → baixo)
45deg (ângulo em graus)

cor1, cor2, ... → as cores que formarão o gradiente (pode ter mais de duas).

A propriedade cursor define qual tipo de ponteiro do mouse será exibido quando o usuário passar sobre um elemento. Normalmente, usamos valores como pointer (mãozinha) ou default (seta), mas também é possível usar uma imagem personalizada como ponteiro.

Como escreve o códico :cursor: url('imagem.png'), tipo-fallback;

url('imagem.png') → caminho para a imagem do cursor.

tipo-fallback → tipo de cursor padrão caso a imagem não carregue (ex: auto, pointer).

A propriedade text-shadow adiciona uma ou mais sombras ao texto. Quando usamos cores vivas e várias camadas de sombra, conseguimos criar um efeito de neon — aquele brilho típico de letreiros luminosos.

Sintaxe Básica :
text-shadow: deslocamentoX deslocamentoY raio-desfoque cor;

deslocamentoX → posição horizontal da sombra (positivo = direita, negativo = esquerda).

deslocamentoY → posição vertical da sombra (positivo = baixo, negativo = cima).

raio-desfoque → define o quão espalhada será a sombra (quanto maior, mais suave).

cor → cor da sombra.

importante avisar que o fundo preto foi um comando adcional para que fosse melhor de perceber

esse código é mais difícil comparados aos outros, mas vamos lá. O movimento de rotação pode servir para chamar atenção, ter engajamento nos objetos ou por opção de aparencia

@keyframes girar {
to { transform: rotate(360deg); }
}

div {
width: 100px;
height: 100px;
background: red;
animation: girar 2s linear infinite;
}

@keyframes girar → define a animação chamada girar, que vai de um estado inicial (0°) até um estado final (rotate(360deg)).
transform: rotate(360deg) → gira o elemento em 360 graus.

animation → aplica a animação ao elemento:
2s → duração de 2 segundos por volta.
linear → mantém a mesma velocidade durante todo o giro.
infinite → repete para sempre.

clique nessa área

Você pode usar transition em quase qualquer propriedade que muda, como cor, tamanho, posição, opacidade, sombra, etc.

Sintaxe : transition: all 0.3s ease;

all → indica que todas as propriedades que mudarem serão animadas.
0.3s → duração da animação (0,3 segundos).
ease → tipo de aceleração da animação (começa devagar, acelera e termina devagar).

passe o mouse nessa área

usamos para aplicar estilos quando o mouse passa sobre um elemento. É muito útil para criar feedback visual ao usuário, deixando a interface mais interativa e intuitiva.

O exemplo que mostrarei é que quando passar o mouse em cima o objeto rotacione (bom dar uma olhada no card de rotação e transition antes)

Sintaxe :
.nome_do_elemento:hover {
transform: rotate(45deg);
}

.nome_do_elemento tem transition: transform 0.5s ease; → isso faz a rotação acontecer suavemente em 0,5s.
.caixa:hover aplica transform: rotate(45deg); → rotaciona o elemento 45 graus quando o mouse passa por cima.
Quando o mouse sai, o elemento volta à posição original, graças ao transition.
