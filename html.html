Glossário

Essa família de tags são designadas á escrita. Elas tem tamanhos diferentes e caracteristicas de margem e sublinhamento algumas. São organizadas sendo <h1> a maior, <h6> menor, em escala com as outras

A tag <h1> é geralmente usada para o título principal da página. Ela representa o conteúdo mais importante e costuma aparecer apenas uma vez no topo do site. Por exemplo, em um artigo, o <h1> pode ser o título do tema principal.

A tag <h2> é usada para dividir o conteúdo em seções importantes. Ela representa um nível abaixo do <h1> e é ideal para marcar sessões como "Introdução", "Exemplos" ou "Benefícios", dentro do conteúdo principal.

A tag <h3> representa um subtítulo dentro de uma seção marcada com <h2>. Por exemplo, se <h2> for “Exemplos de uso”, o <h3> pode ser algo como “Exemplo com parágrafos” ou “Exemplo com imagens”.

A tag <h4> é usada como um subtítulo dentro de uma seção marcada com <h3>. Ela ajuda a organizar conteúdos que têm muitos detalhes ou subdivisões, mantendo a hierarquia clara sem confundir o leitor.

A tag <h5> representa um nível ainda mais específico de conteúdo. É útil quando você precisa dividir uma seção <h4> em partes menores, como em tutoriais longos, documentações técnicas ou manuais.

A tag <h6> é o título de menor peso e importância dentro da hierarquia de títulos HTML. Ela raramente é usada, mas pode ser aplicada em situações bem específicas, como referências internas, notas secundárias ou estruturação profunda de conteúdo.

define um hiperlink, permitindo que você crie links para outras páginas web, arquivos, locais dentro da mesma página, ou até mesmo para endereços de e-mail. Ela é uma das tags mais fundamentais em HTML e é amplamente utilizada para a navegação entre diferentes conteúdos.

    O atributo href é essencial para definir o destino do link
    Ele pode conter:
  • URLs: Endereços de outras páginas web (ex: href="https://www.example.com").

  • Âncoras: Links para seções específicas dentro da mesma página (ex: href="#secao1").

  • Arquivos: Links para download de arquivos (ex: href="documento.pdf").

  • Endereços de e-mail: Links que abrem o cliente de e-mail padrão (ex: href="mailto:email@example.com").

O conteúdo que você escreve define o texto visível do link, que o usuário clica para navegar para outro local. pode ser um botão também usando outras tags juntos

Os botões sem duvidas é uma das coisas mais comuns em sites, curiosidade que todas as abas com informações que você abre aqui são feitas com a tag <button>

  • <button type="button">
    Função: Um botão genérico que não envia formulários. Usado para: Ações com JavaScript (como abrir um modal, alternar um menu, etc).

  • <button type="submit">
    Função: Envia o formulário onde o botão está inserido. Comportamento padrão se type for omitido.

  • <button type="reset">
    Função: Reseta os campos do formulário para seus valores iniciais.
